This project uses laptop webcam running Python to look at fingers 
then turn on seven segment disply connected to Arduino according to 
number of fingers detected from 1 - 5.

Hardware Spec:
- Laptop
- Webcam
- Arduino
- USB cable
- Seven Segment Display
- Resistors (220ohms x 8)

Software:
- Windows 10
- Python3

packages:
- openCV
- pySerial
mediapipe

Run:
- load cv_with_ssd.ino code to Arduino
- run FingerCountingProject.py program.
